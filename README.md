# tablature.io

Uma ferramenta gratuita para escrever e compartilhar ritmos de bateria de forma simples e visual, direto pelo navegador. Sem instalação, sem cadastro.

---

## Para quem é isso?

Para bateristas, percussionistas, professores de música e qualquer pessoa que queira **anotar um ritmo** sem precisar saber ler partitura.

Se você já anotou um groove em um papel rascunho, o tablature.io faz isso de forma limpa, organizada e fácil de compartilhar.

---

## O que é uma tablatura de bateria?

É a forma mais prática de escrever música para bateria. Em vez de uma partitura tradicional (que exige anos de estudo para ler), a tablatura usa uma grade simples:

- **Linhas horizontais** — cada linha representa uma peça da bateria (bumbo, caixa, hi-hat, etc.)
- **Símbolos na linha** — indicam quando aquela peça deve ser tocada

Qualquer baterista reconhece esse formato. É o mesmo que circula em fóruns, grupos de WhatsApp e livros didáticos.

---

## Como funciona

### Escolha a peça que quer adicionar

Na barra de instrumentos, você verá todas as peças da bateria disponíveis — desde o bumbo até os pratos. Clique em uma delas para selecioná-la.

### Clique na grade para colocar uma nota

Depois de selecionar a peça, clique no ponto do ritmo onde ela deve soar. A nota vai se posicionar automaticamente no tempo certo, sem você precisar se preocupar com alinhamento.

### Ajuste o que quiser

- Arraste uma nota para mudar de lugar
- Clique em uma nota para deletá-la
- Use **Ctrl+Z** para desfazer qualquer erro

### Adicione quantos compassos precisar

Seu groove tem mais de um compasso? Basta clicar em **"+ Compasso"** para expandir a tablatura.

### Ouça o ritmo

Clique em **Play** para ouvir sua tablatura tocando. Cada peça reproduz o som correspondente, facilitando conferir se o ritmo está do jeito que você imaginou.

### Exporte e compartilhe

Quando estiver pronto, baixe sua tablatura como **PDF** para imprimir ou enviar, ou como **imagem** para postar onde quiser.

---

## Não perde nada

Tudo que você fizer é salvo automaticamente. Se fechar o navegador e abrir de novo, sua tablatura vai estar lá esperando por você.
